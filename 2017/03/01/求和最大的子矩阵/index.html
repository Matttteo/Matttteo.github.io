求和最大的子矩阵.md

在微软面试的时候被问道一个题目，给一个m乘n的矩阵，矩阵内为一些值，现在求一个所有元素和最大的子矩阵。如矩阵
4 2 5
3 -4 5
2 1 -10
元素和最大的子矩阵为：
4 2 5
3 -4 5

首先考虑矩阵为一维的情况，我们的矩阵现在是一个数组$A = { a_1 , a_2, … ,a_n }$，现在要输出的是最大的子数组$a_i ~ a_j$。我们设置一个滑动窗口，窗口的左端x和右端y初始都设为下标为零处，我们再使用一个值res标记目前出现的，和最大的窗口值。然后我们从零开始迭代滑动窗口的右侧下标y。

  1. 当$a_x + .. + a_y = sum> 0$ ，取$res = max(res, sum)$，$y = y + 1$;
  2. 当$a_x + .. + a_y = sum <= 0$, $x = x + 1; y = y + 1$
  3. 当 y > n时，结束循环，最大值为res。
    在上面这个过程中，可以很方便的将窗口左右端也保存，这样就可以找到最大子数组的具体位置了。这个算法的时间复杂度为$O(n)$。

将一维情况推到二维矩阵$A = {a_11, a_12, …, a_1m, …., a_n1, a_n2, … , a_nm}$，如果使用暴力法，显然算法复杂度高达$O(n^4)$。
我们考虑利用一维的结果，首先我们计算一下所有的$r_i ~ r_j$（$1 <= i, j <= n$ )行的矩阵和，其结果为一个数列$B_ij = {b_1, b_2, …,b_m}, 其中b_k = aik + a{(i+1)k} + … +a_jk$。获得$Bij$后，我们在上面执行一维的算法，找出最大的子数列$B{ij}[x ~y]$及其值$res_ij$这时候我们就找到了以$i, j$为行边界条件下元素和最大的一个子矩阵，与其值$res_ij$，我们只需要遍历所有$i, j$的组合，就可以找到全局和最大的子矩阵与其值$res$了。
