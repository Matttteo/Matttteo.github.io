The same repeated number may be chosen from C unlimited number of times.

Note:

All numbers (including target) will be positive integers.
The solution set must not contain duplicate combinations.

For example, given candidate set [2, 3, 6, 7] and target 7,
A solution set is:

[ [7], [2, 2, 3]]

遐想

晚上路过新开的山东水饺店，没忍住点了份带回寝室吃。最近一些新的饭店开了，东西都还算好吃，老板刚开始干干劲也很十足的样子。我站在那里想，当初他们怎么想来这里开这个餐馆的啊。

”你看看我做的水饺好吃不，能吸引顾客吗？好吃的话我们去上海开个店吧，正好我认识有同乡开过，可以问问他。开店需要点人手，亲戚老乡家的孩子现在也没什么正经工作就叫他们吧，跟着我他们家长也会放心吧。之前积蓄这下要拿出来大部分可能还要借钱，和老婆商量商量，大概她会同意吧。上海那里人多消费能力高，干的好了比待在老家挣的肯定多，反正过年还是会回家的，干得好了比现在窝在家里也有面子些。干的不好了大不了回去再慢慢积累吧，反正再苦再累也比什么也不干强，孩子马上也长大了，为了他们也要好好拼一把。“

想着想着我的那份就做好了，然后我回去就把它吃了。
