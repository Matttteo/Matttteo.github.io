前言

这个东西是nju计算机系统基础，也就是计算机组成原理课的PA，内容主要是要实现一个类似qemu简化版的nemu。要做的内容见其文档，主要要做的东西在这个图下面。

内核部分暂时不用自己写，主要是虚拟出来一个cpu，自己去完成指令的译码执行等。虚拟的魔术主要发生在：

  1. 模拟了寄存器，并且按照汇编级的代码去运行程序。也就是寄存器搬来搬去，cpu去完成逻辑运算，但是其实逻辑运算还有寄存器的赋值运算等还是要用c语言实现。
  2. 虚拟了内存出来，其实内存就是一个大的数组而已。
  3. 以后想到了再说。

这个实验比较好玩的地方在于，比写内核还要更底层一点，因为写内核你不用去关心指令是怎么执行的，你只要按照cpu的指令集用去实现一些库函数，然后实现内存管理、内存堆栈的管理，线程管理等。同时你还要关心MMU，cache，内存管理（这点和编写内核类似了），还有一些外部设备。实现了这个，你可以让一个内核跑在这个上面（其实一些内核的课就是在qemu上跑，这个就是去实现一个qemu），所以还是挺有意思的。

一，安装环境

文档里使用Docker作为虚拟机，这比用vmware要轻量级许多。有关docker，可以看作是一个操作系统层次的虚拟化技术。传统虚拟机技术是虚拟出一套硬件，在上面运行整个操作系统，在这个系统上运行进程。而Docker内的应用程序进程直接运行在Host的内核。有关Docker的介绍，见[1]，[2]。


安装过程大致为：

  1. 在系统上安装Docker
  2. 编写Dockerfile
  3. build 系统镜像
  4. create 一个container（container名字，docker镜像，将ssh端口和寄主的20022端口绑定
  5. docker start ics-vm / docker stop ics-vm
  6. 使用ssh连接

遇到的坑：
主要是build的时候，在docker里apt-get 需要先update一下，最好改一下源。另外如果update失败先单独update 一次，否则重复执行docker只会用cached的数据，结果安装其他软件还是找不到。当然我最后解决的方法不是改源，我设置了代理。。

因为我是在我的ubuntu上写的，所以其实没有用docker，并且我用sublime不用vim，所以安装到这里就结束了

求和最大的子矩阵

Posted on 2017-03-01   |  

在微软面试的时候被问道一个题目，给一个m乘n的矩阵，矩阵内为一些值，现在求一个所有元素和最大的子矩阵。如矩阵
4 2 5
3 -4 5
2 1 -10
元素和最大的子矩阵为：
4 2 5
3 -4 5

首先考虑矩阵为一维的情况，我们的矩阵现在是一个数组$A = { a_1 , a_2, … ,a_n }$，现在要输出的是最大的子数组$a_i ~ a_j$。我们设置一个滑动窗口，窗口的左端x和右端y初始都设为下标为零处，我们再使用一个值res标记目前出现的，和最大的窗口值。然后我们从零开始迭代滑动窗口的右侧下标y。

  1. 当$a_x + .. + a_y = sum> 0$ ，取$res = max(res, sum)$，$y = y + 1$;
  2. 当$a_x + .. + a_y = sum <= 0$, $x = y + 1; y = y + 1$
  3. 当 y > n时，结束循环，最大值为res。
    在上面这个过程中，可以很方便的将窗口左右端也保存，这样就可以找到最大子数组的具体位置了。这个算法的时间复杂度为$O(n)$。

将一维情况推到二维矩阵$A = {a_{11}, a_12, …, a_1m, …., a_n1, a_n2, … , a_nm}$，如果使用暴力法，显然算法复杂度高达$O(n^4)$。
我们考虑利用一维的结果，首先我们计算一下所有的$r_i ~ r_j$（$1 <= i, j <= n$ )行的矩阵和，其结果为一个数列$B_ij = {b_1, b_2, …,b_m}$, 其中$b_k = a_ik + a_{(i+1)k} + … +a_jk$。获得$Bij$后，我们在上面执行一维的算法，找出最大的子数列$B\{ij}[x ~y]$及其值$res{ij}$这时候我们就找到了以$i j$为行边界条件下元素和最大的一个子矩阵，与其值$res\{ij}$，我们只需要遍历所有$i j$的组合，就可以找到全局和最大的子矩阵与其值$res$了。
在计算$B_ij$的时候，我们利用之前的计算结果即可。

我所不能了解的事

从繁华的上海，大车转小车，回到了我成长的家。和家人聊聊天，说说发生的事情，居然感觉自己也是个蛮可爱的人了。回乡的见闻年年如此没什么特别，房价变得贵了但是和自己也没什么关系，城里乡里建设的都变得更好了，马路修的也越来越像样子，各种工程项目在准备上马，高铁马上要开通了。国家像是一个大锅，中间的沸腾起来了，旁边的也带动着翻滚，趁着这个热火劲，能多做就多做，大概是这个时代的主题。

聪明人知道在这个时代抓住机会，同时也为下一个寒冬做点准备。谁知道水什么时候会冷却呢，一个平面的社会也很难抵抗风险，底线还是得靠政府维持。不过，为这个事情瞎操心的人注定不适合生活在这个时代里，当你有机会给自己给社会带来实打实的利益的时候就不要考虑什么理想和远景了。

当然在这个热火的时代之下也有一些难看的东西，高房价给人巨大的压力，爹学成功成为你国第一学，年轻人应了某厚书记说的，“靡靡之音”，很弱，不硬朗了，喜欢逼逼，喜欢怀念，喜欢沉迷幻想。当然这一切也都是历史的进程，专门去攻击这个时代的人是非常短见的，不管是直男癌，拜金女，不要去轻易的攻击，特别是当你不知道对方经济情况的时候。

分享一个罗大佑第三张专辑，这张很民谣。流行音乐不是艺术品，当然是主要为了迎合这个时代人的审美，所以你看到那些形容浓的化不开的纠结情绪的歌曲大行其道不是没有原因的。这张专辑迎合的是哪些人呢，我不知道，因为我已经不是那个时代的人，不过我觉得的它很清新，没有自怨自艾。

Oasis, Radiohead, Blur, The stone roses

Posted on 2016-11-21   |  

这几个乐队是我目前最喜欢的几个乐队，凑巧都是英伦摇滚，我也不是一种只听某种口味的乐迷，只是英伦摇滚确实是好听。

首先推荐的是Oasis的《definitely maybe》，里面好听的太多了，可以反复重复听好几遍。特点是简单直接，旋律动人，满满的少年心气。Liam 的嗓音就像一个孩子一样，Noel在这个首专里没有唱，其实Noel心里也是很柔软的。Noel说他和Liam唱supersonic的时候体验是兄弟一条心的感觉，但是在现实中简直你死我活，也是非常奇怪。

然后是Radiohead，主唱Yorke是一个看上去有点神经质的天才。Radiohead比Oasis好的一点是他们不乱搞，可以静下来搞一些概念的东西。《OK, computer》这张专辑是我第一次接触radiohead听的，这张专辑有一个完整的概念，同时又不是那种跳大神或者太先锋的音乐，单纯的把这种专辑当成一个摇滚专辑听你也不会失望。这种感觉就像听The beatles的《Sgt. Pepper’s Lonely Hearts Club Band》。

Blur和Oasis是同一时期的两个不相上下的乐队，可能Blur比Oasis“有出息”一点，曲风更多变，又不像Radiohead那样阴郁，弱点在那种优美的旋律少了点，但是也不是没有，tender， you are so great就很好听，还有新专辑里的ong ong。抛却掉商业上的包装，blur的音乐就是那种各个方面都很好的摇滚乐，优质生，不脏，一上来你可能觉得不是那么好听，但是只要稍稍进入点状态就感觉他们确实很厉害。推荐他们的同名专辑《blur》。

The stone roses比之前几个乐队要早一点，说影响了后来的乐队一点也不夸张。一开始我是听惯了Oasis和blur去听石玫瑰的，感觉好像不够爆炸，直到最近我听了Oasis的lost tape，感觉似曾相识，好像挺像石玫瑰的，然后我又去听了《The Stone Roses》，这次就真的感觉爆炸了，各方面的冲击，觉得特别好。你考虑到他们还是自己搞出来的，搞出自己的味道，以前从来没有的，就更觉得厉害。

当从这几张专辑里走出来，再去听一些模仿的，没特点的，不好听更谈不上爆炸的乐队，真的感觉他们差距挺大的。

断舍离

Posted on 2016-11-19   |  

知乎上看到一个问题，如何让自己快速的静下来？有人回答收拾房间，这正符合我。我不是一个整整齐齐的人，但是我也喜欢整理房间，当东西妥妥当当，仿佛我会忘记一些烦恼事，让自己继续前进下去，似乎我又掌控了我的人生。

今天把自己的书桌整理了一下，已经是乱的不成样子，但是我之前竟然没有察觉。整理他花了一个多小时，其间丢了很多东西，也算是一种断舍离吧。记得高中的时候每次整理完屋子，丢了一些东西后，那种“断舍离”的感觉会强烈一点，一次我说给同学听，同学表示不理解，为什么要连自己的小工艺品也丢了呢，我说这个东西寄托了我无聊时间太多的情绪，丢了他我心情很轻松。于是这样我就可以向前了，于是我更无拘无束了，但是隐隐约约又觉得有所失，后来我又不太爱丢东西了。

其实连青春都是留不住的。走在冬天的路上，路边的冰用力也踩不破，过了两天居然连水也没留下了。更不用说那些草本植物，一茬换一茬。大自然倾向于替换，我们也是要被替换的东西。所以把旧的替换成新的，这是再正常不过的事情了，这样整个体系才能正常运转。

在我整理桌子时，发现居然还留着新生报道的手册，顺便又回想了一下自己怎么来的，然后把它丢了。恩，问题来了，丢了这个东西为什么我脑子里还在想呢，就像你搬到了异国他乡身边没有一个东西关于你的亲人，为什么你每时每刻都会想到他们呢，你被关到了监狱里，除了一个衣服身边什么都没有，为什么你还是记挂着自己的老婆孩子呢？物质的环境是一种线索和提醒，比如一辆车提醒了我可以驾车去很多地方，一个舒服的床是接下来我有一个好的睡眠的线索，一个整洁的环境常常能提醒自己凡事安排妥当减轻未知的焦虑，等等。而这些提醒之所以成立，也在于我们心里已经有了这些概念，当物质的线索没了，那些真正的渴望的东西会在精神里找线索。所以整理屋子，实际上是扫除那些令人不快的线索，从这个角度看，也不完全是体力活了。

因此，断舍离，不是在丢东西，而是一种心理暗示，我很轻松，我丢掉了包袱，被丢了的东西产生的空缺作为这种心里暗示在现实中的线索。

据说乔布斯的屋子开始一直很空荡荡，因为他只选自己喜欢的东西进来，这算是一种禅境了。
